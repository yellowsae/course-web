### Axios 请求模块 

```ts
// 已经封装 axios 
//api/index.js


// 按照请求类型对axios进行封装
const api = {
  get(url, data) {  // data 没有可以不
    return instance.get(url, { params: data })
  },
  post(url, data) {
    return instance.post(url, qs.stringify(data))
  },
}


```
